Funktioner 2018-07-14
- 

* Individåtgärder
* Fordons meny
* Stänga av motor! (Fordonet MÅSTE stå stilla!)
* Öppna dörrar
* Öppna backlucka
* Öppna MotorHuv
* Veva ner fönster (Funkar bara "LOKALT")
* Farthållare
* Byt säte
* Idkort
* Kolla ID-Kort
* Visa ID-kort
* Kissa
* Bajsa
* Sätt av/på ögonbindel

Det är också inlaggd en funktion att man kan klicka på B för att ta på sig bälte!
